# Quotable

A minimal, beautiful random quote generator with dark/light mode.

**[Live Demo](https://div1000.github.io/random-quote-generator/quote-generator.html)**

---

## Features

- Fetches random quotes from the [DummyJSON Quotes API](https://dummyjson.com/docs/quotes) — no API key required
- One-click copy to clipboard
- Dark / Light theme toggle
- Smooth fade transitions between quotes
- Fully responsive — works on mobile and desktop
- Zero dependencies, single HTML file

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- [DummyJSON API](https://dummyjson.com/quotes/random) for quotes
- [DM Serif Display + DM Sans](https://fonts.google.com) via Google Fonts

## Getting Started

No build step needed. Just open the file in a browser:

```bash
git clone https://github.com/div10000/random-quote-generator.git
cd random-quote-generator
open index.html
```

Or serve it locally:

```bash
npx serve .
```
